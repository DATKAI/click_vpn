Адрес, по которому сервис раздачи ссылок (/s/) доступен сотрудникам снаружи. Обычно https://<WAN_IP>:<порт_nginx> (порт показал install-share-service.sh). Если пусто — берётся адрес текущего запроса (для локалки).

Импорт клиентов из CSV

Массовое создание клиентов из файла

Формат CSV (первая строка — заголовки):
username,full_name,email,org,valid_days,password ivanov,Иванов Иван,ivanov@co.ru,ЗПК,365, petrov,Петров Пётр,petrov@co.ru,ЗПК,730,secret
  • username — логин (латиница), обязательно
  • org — точное название организации, обязательно
  • full_name, email, password — необязательно
  • valid_days — срок (по умолчанию 365)
  • Сервер берётся из организации автоматически

Новый клиент

Используется как CN сертификата. Только латиница, без пробелов.

Сервер:
У организации нет серверов. Привяжите её к серверу.
OpenVPN: будет выдан файл .ovpn с сертификатом. WireGuard: ключи сгенерируются автоматически, профиль .conf. Пароль не нужен. IKEv2: вход по логину и паролю. Логин = имя пользователя. Профиль .mobileconfig (Apple) или ручной ввод.
