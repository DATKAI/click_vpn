Импорт клиентов из CSV

Массовое создание клиентов из файла

Формат CSV (первая строка — заголовки):
username,full_name,email,org,valid_days,password ivanov,Иванов Иван,ivanov@co.ru,ЗПК,365, petrov,Петров Пётр,petrov@co.ru,ЗПК,730,secret
  • username — логин (латиница), обязательно
  • org — точное название организации, обязательно
  • full_name, email, password — необязательно
  • valid_days — срок (по умолчанию 365)
  • Сервер берётся из организации автоматически
